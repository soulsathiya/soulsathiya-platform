
# SoulSathiya Platform

AI-driven matrimonial and compatibility platform designed to create deeply compatible and fulfilling marriages.

## Overview

SoulSathiya combines:

- Psychometric compatibility matching (36-question assessment)
- Deep couple exploration (108-question pair analysis)
- AI-driven matchmaking and insights
- Tiered subscriptions and premium features
- Admin moderation and analytics

## Platform Structure

frontend/ – React UI  
backend/ – FastAPI services  
models/ – data models  
services/ – compatibility & deep exploration engines  
memory/ – prompts & build context  
tests/ – test suites  

## Key Features

- Intelligent match compatibility scoring  
- Deep couple compatibility reports  
- Messaging & interaction system  
- Subscription & boost monetization  
- Admin dashboard & moderation  
- Notifications & demo reports  

## Vision

SoulSathiya aims to increase the number of happy, deeply compatible marriages by combining psychology, AI, and relationship science.
